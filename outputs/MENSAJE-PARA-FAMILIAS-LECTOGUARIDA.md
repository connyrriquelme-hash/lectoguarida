# Mensaje para las familias

Estimada familia:

Para ingresar a Lectoguarida desde el celular o computador, abra este enlace:

https://lectoguarida-conny.connyrriquelme.workers.dev/

Código del curso: `GUARIDA-27`

El estudiante debe buscar su primer nombre y apellido. La primera vez creará y repetirá un PIN personal de 4 números; luego podrá entrar a su mapa de 105 niveles. Si el PIN presenta alguna dificultad, debe comunicarse con Conny para recibir uno nuevo sin perder avances. Los resultados llegarán automáticamente al panel docente.

No es obligatorio instalar nada. Se recomienda usar Chrome y permitir el micrófono solamente durante las actividades de lectura.
